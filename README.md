Handwritten Digit Recognition using CNN

This project implements a Handwritten Digit Recognition System using Convolutional Neural Networks (CNN) with TensorFlow and Keras. The model is trained on the MNIST dataset along with custom handwritten digit images to accurately classify digits from 0 to 9.

Features
Handwritten digit classification (0–9)
Deep Learning based CNN architecture
MNIST dataset integration
Custom dataset support
Data augmentation for improved performance
Model evaluation with accuracy and classification report
Prediction on new handwritten images
Technologies Used
Python
TensorFlow / Keras
OpenCV
NumPy
Scikit-learn
Google Colab
Results

The model achieves high classification accuracy on handwritten digit recognition tasks and can predict unseen digit images with confidence scores.
